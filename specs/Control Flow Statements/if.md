If statements operate over comparisons, Booleans, and optional variables.
```
//Operate on booleans
if(true){
	printf("to always run");
}

//operate on comparisons
if(1==1){
	printf("to always run");
}

//operate on optional variables
var ?u8: no = null;
if(no){
	printf("not null");
}else{
	printf("null!!!");
}
```
